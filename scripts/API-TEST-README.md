# 和风天气API测试工具使用指南

## 📋 工具概述

本目录包含了完整的和风天气API测试工具链，帮助您验证API配置和功能是否正常工作。

## 🛠️ 可用工具

### 1. JWT验证工具
- **文件**: `verify-jwt.js` / `verify-jwt.bat`
- **功能**: 验证JWT Token的格式、有效性和基本API调用
- **适用场景**: 快速验证JWT配置是否正确

### 2. API功能测试工具
- **文件**: `api-test.js` / `api-test.bat`
- **功能**: 全面测试所有天气API端点的功能
- **适用场景**: 完整的API功能验证和性能测试

## 🚀 快速开始

### 前置条件
1. 确保已安装 Node.js
2. 配置好 `.env.local` 文件
3. 确保JWT Token有效且未过期

### 运行测试

#### Windows用户（推荐）
```bash
# 进入scripts目录
cd scripts

# 运行JWT验证
verify-jwt.bat

# 运行完整API测试
api-test.bat
```

#### 跨平台用户
```bash
# 进入scripts目录
cd scripts

# 运行JWT验证
node verify-jwt.js

# 运行完整API测试
node api-test.js
```

- 📅 **3天天气预报** - 短期天气预测
- 📊 **7天天气预报** - 中期天气预测

```

```

### 常见错误码
- **401**: JWT Token无效或过期
- **402**: 超过请求量限制
- **403**: 无权限访问该API
- **404**: 请求的资源不存在
- **429**: 请求过于频繁

## 🔧 故障排除

### 1. JWT Token相关问题
```bash
# 重新生成JWT Token
generate-jwt.bat

# 验证新生成的Token
verify-jwt.bat
```

### 2. 网络连接问题
- 检查防火墙设置
- 确认网络连接正常
- 尝试使用代理（如需要）

### 3. 配置文件问题
- 检查 `.env.local` 文件是否存在
- 验证环境变量格式是否正确
- 确认API Host地址是否正确

### 4. API限制问题
- 检查和风天气控制台的使用量
- 确认账户状态正常
- 验证API权限设置
