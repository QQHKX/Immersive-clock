## 🌟 功能特点

### 🕐 核心模式
- **时钟模式** - 优雅的全屏时钟显示

## 📱 使用指南

### 基本操作
- **显示/隐藏控制界面**：点击屏幕任意位置或按空格键
- **切换模式**：点击顶部的模式选择器
- **全屏切换**：点击控制栏中的全屏按钮
- **打开设置**：点击左下角的设置按钮

### 倒计时模式
- **设置时间**：双击时间显示区域或点击「设置」按钮
- **预设时间**：选择常用的时间段（10分钟、30分钟、1小时等）
- **开始/暂停**：点击控制栏中的播放/暂停按钮
- **重置**：点击控制栏中的重置按钮
- **结束提示**：倒计时结束时播放提示音

### 秒表模式
- **开始计时**：点击开始按钮
- **暂停/继续**：点击暂停按钮
- **重置**：点击重置按钮清零
- **里程碑**：自动记录重要时间节点

### 晚自习模式
- **高考倒计时**：显示距离目标年份高考的剩余天数
- **设置目标年份**：在设置面板中修改高考年份
- **课程表管理**：
  - 添加学习时段：点击「添加」按钮
  - 编辑时段：点击时段项进入编辑模式
  - 删除时段：点击删除按钮
  - 重置课程表：恢复默认设置

### 设置面板
- **目标年份**：设置高考目标年份（当前年份+1到+10年）
- **噪音校准**：
  - 自动校准：点击「重新校准」进行3秒环境音量检测
  - 手动调节：使用滑动条调整校准值（20-100dB）
  - 清除校准：重置校准数据
- **课程表设置**：管理学习时段和课程安排
